# celo-dice-roll-contracts

Solidity smart contracts for the Dice Roll dApp on Celo blockchain.

## Contracts

- `DiceRollV2.sol` — Core game logic with on-chain randomness

## Development

```bash
npm install
npx hardhat compile
npx hardhat test
```

## Deployment

Contracts are deployed on Celo mainnet.
